# KeenKeeper — Keep Your Friendships Alive

## 1. What is the name of your project?

KeenKeeper — Keep Your Friendships Alive


## 2. What is a short description of your project?

KeenKeeper is a friendship tracking web application that helps users keep their friendships active and organized. Users can view friend cards, check detailed friend information, log interactions like call, text, and video, and see friendship activity on a timeline and stats page.


## 3. What technologies did you use?

- React.js
- React Router DOM
- Tailwind CSS
- Recharts
- React Icons
- JSON file for friend data


## 4. What are 3 key features of your project?

### 1. Friend Management
The home page shows all friends in card format using data from a JSON file. Each card includes picture, name, days since contact, tags, and status.

### 2. Friend Details and Quick Check-In
Each friend has a details page where users can see full information, relationship goals, and quick action buttons for Call, Text, and Video.

### 3. Timeline and Analytics
The app automatically adds interaction history to the timeline page and also shows friendship analytics on the stats page with a pie chart for call, text, and video interactions.
